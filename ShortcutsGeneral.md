# -------------- GENERAL SHORTCUTS --------------

## Ctrl shortcuts

* ! Select All Items - Ctrl + A

* ! Copy - Ctrl + C
* ! Paste - Ctrl + V
* ! Cut - Ctrl + X

* ! Undo - Ctrl + Z
* ! Redo - Ctrl + Y OR Ctrl + Shift + Z

* ! Save - Ctrl + S
* ! Save As - Ctrl + Shift + S

* ! Print - Ctrl + P

* ! Find - Ctrl + F
* ! Find and Replace - Ctrl + H

* ! Open - Ctrl + O
* ! Close - Ctrl + W

* ! New - Ctrl + N
* ! New Tab - Ctrl + T
* ! New Window - Ctrl + Shift + N
