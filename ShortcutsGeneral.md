# -------------- GENERAL SHORTCUTS --------------

## Ctrl shortcuts

* ! Select All Items - Ctrl + A

* ! Copy - Ctrl + C
* ! Paste - Ctrl + V
* ! Cut - Ctrl + X

* ! Undo - Ctrl + Z
* ! Redo - Ctrl + Y OR Ctrl + Shift + Z

* ! Save - Ctrl + S
* ! Save As - Ctrl + Shift + S

* ! Find - Ctrl + F
* ! Find and Replace - Ctrl + H

* ! Close - Ctrl + W

* ! New Tab - Ctrl + T

* ? New - Ctrl + N

* ? Print - Ctrl + P

* ? Open - Ctrl + O

* * New Window - Ctrl + Shift + N
